# 📊 Amazon Stock Analysis Dashboard (Microsoft Excel)

## 📌 Project Overview

This project presents an interactive **Amazon Stock Analysis Dashboard** built in **Microsoft Excel**. The dashboard analyzes Amazon's historical stock market performance using Pivot Tables, Pivot Charts, Excel formulas, and interactive slicers.

The objective of this project is to transform raw stock market data into meaningful insights that help understand price trends, trading volume, and overall market performance.

---

## 🚀 Features

- 📈 Interactive Dashboard
- 📅 Year-wise Stock Performance Analysis
- 📊 Pivot Tables & Pivot Charts
- 🎛️ Dynamic Slicers for Data Filtering
- 📉 Stock Price Trend Analysis
- 📦 Trading Volume Analysis
- 📍 KPI Cards for Key Metrics
- 🎨 Professional Dashboard Design

---

## 📂 Dataset Information

The dataset contains historical Amazon stock market data including:

- Date
- Open Price
- High Price
- Low Price
- Close Price
- Adjusted Close Price
- Trading Volume

## 📊 Dashboard KPIs

The dashboard provides insights such as:
